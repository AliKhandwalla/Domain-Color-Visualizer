# Domain-Color-Visualizer
This is a project for visualizing graphs on the imaginary plane, where you can see imaginary roots of different functions that you can plot.

## Features
- Plots complex and imaginary numbers
- Supports color maps
- Supports most functions

  ## Installation
  1. Clone the repository
  ```bash
  git clone https://github.com/AliKhandwalla/domain-color-visualizer.git
  cd domain-color-visualizer
  ```
  2. Install the dependencies
  ```python
  pip isntall -r requirements.txt
  ```
  3. And finally, run the python file and enter any function
  ```python
  python domain-color-visualizer.py
  ```
  4. See your graph on the real-complex plane!
